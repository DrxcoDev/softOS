# SoftOS

Is a simple operative system with a simple bootloader and kernel

# Requisites
  -  `GCC`
  -  `NASM`
  -  `WSL (IF YOU USE WINDOWS)`
  -  `QEMU`

# Start
1. `clone this repository`
2. Input: `./build.sh`

- You will can look a green text.

## Only Memory
1. `cd util`
2. Input: `./memory.sh`
